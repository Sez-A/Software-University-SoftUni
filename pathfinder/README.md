# Pathfinder 

## Pathfinder application built for [Spring-Advanced course](https://softuni.bg/trainings/3494/spring-advanced-october-2021#lesson-31208) at SoftUni

## Description 
#### Pathfinder is a web application that allows you to find a new route for tourism and enjoy the nature. With pathfinder you are able to choose between pedestrian, bicycle, motorcycle or car routes. When logged in, you can read the most important information about routes, see pictures, video and difficulty level of route also you can leave a comment.
